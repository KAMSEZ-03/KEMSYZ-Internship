# Khurram Portfolio App

A Flutter-based portfolio application showcasing projects, skills, and personal information. Designed for cross-platform deployment (Android, iOS, Web, Windows, macOS, Linux).

---

## Features

- Fully responsive design for web, mobile, and desktop  
- Home page with hero section and personal introduction  
- About section highlighting skills, experience, and education  
- Projects section showcasing completed work  
- Services section (optional) to display expertise  
- Contact section with email and social links  
- Footer with social links and copyright notice  
- Smooth navigation using a custom navbar  
- Modular architecture for easy maintenance  

---

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)  
- [Dart SDK](https://dart.dev/get-dart)  
- VS Code, Android Studio, or any preferred IDE  

### Run Locally

```bash
git clone https://github.com/KAMSEZ-03/KEMSYZ-Internship.git
cd Portfolio_Code_Week_1_and_2/portfolio_app
flutter pub get
flutter run
```
